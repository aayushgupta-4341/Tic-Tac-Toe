# ❌⭕ Tic Tac Toe Game

An interactive and modern **Tic Tac Toe Game** built using **HTML, CSS, and JavaScript** with a unique **Dark/Light Mode Toggle 🌙☀️**.

---

## 🚀 Live Demo

🔗 https://tic-tac-toe-gamy.netlify.app/

---

## 📂 GitHub Repository

🔗 https://github.com/aayushgupta-4341/Tic-Tac-Toe

---

## ✨ Features

* ❌⭕ Two Player Mode (X vs O)
* 🌙☀️ Dark / Light Mode Toggle
* 🎯 Smart Win Detection Logic
* 🤝 Draw Detection
* 🔄 Restart Game Option
* 📱 Fully Responsive Design
* 🎨 Clean & Modern UI

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
